Jak používat StudyPrep

🎴 Spaced Repetition (SRS)

Aplikace používá algoritmus SM-2 pro optimální rozložení opakování. Kartičky se zobrazují podle toho, kdy je potřeba je zopakovat.

  • Znovu - Kartičku uvidíš znovu za pár minut
  • Těžké - Interval se zkrátí
  • Dobře - Interval se prodlouží (1→3→7→14 dnů...)
  • Snadné - Interval se výrazně prodlouží

⌨️ Klávesové zkratky

  • Space - Otočit kartičku / Zobrazit odpověď
  • 1 - Znovu
  • 2 - Těžké
  • 3 - Dobře
  • 4 - Snadné
  • → - Další kartička (bez hodnocení)

📥 Import/Export

Data můžeš exportovat do CSV a později importovat. Formát: topic,front,back,explanation

🎧 Audio Loop
