Study App Skeleton
==================

Tato složka obsahuje kostru studijní aplikace, kterou lze snadno kopírovat a upravovat pro různé předměty.

Jak vytvořit nový předmět:
1. Zkopírujte celou tuto složku a pojmenujte ji podle nového předmětu (např. "Dejepis").
2. Otevřete soubor 'js/data.js' v textovém editoru (Notepad, VS Code).
3. Upravte data:
   - Změňte 'topic' na název předmětu.
   - Nahraďte obsah pole 'flashcards' vašimi otázkami a odpověďmi.
   - Nahraďte obsah pole 'quizzes' vašimi kvízovými otázkami.
   - Upravte 'topics' pokud chcete změnit kategorie témat.
4. Uložte soubor.
5. Otevřete 'index.html' v prohlížeči.

Není potřeba žádný server ani instalace.
